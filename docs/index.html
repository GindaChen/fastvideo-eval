WanGame Eval

Keyboard-first evaluation tool for AI-generated Minecraft videos.
Matrix comparison, instant ratings, JSONL export — under 20 minutes per checkpoint.

⭐ View on GitHub 📖 Quick Start

Built for Speed

🎬

Card-style Evaluation

Tinder-style swipe through videos. Rate Good/Bad/Skip with single key presses. Configurable 1-4× playback speed.

🔲

Matrix Comparison

Side-by-side grid: rows are prompts, columns are training steps. Compare the same prompt across checkpoints at a glance.

⌨️

Keyboard-First

J/K/L to rate, U/O to navigate, arrow keys for matrix cells, number keys for reasons. Hands never leave the keyboard.

📥

JSONL Storage

Ratings saved to data/ratings.jsonl instantly. Human-readable, Git-friendly. Export and import from the dashboard.

🏷️

Reason Tagging

Tag bad videos with 9 structured reasons (wrong direction, jittery, no movement…). Number keys 1-9 for instant toggle.

📊

Live Scoring

Per-checkpoint scores computed on-the-fly. Dashboard shows committed, skipped, and unrated counts with progress bars.

Get Started in 60 Seconds

1

Clone & install

git clone https://github.com/GindaChen/wangame-eval.git
cd wangame-eval && pip install -e .
2

Get your WandB API key

Go to wandb.ai/authorize and copy your key.

3

Start the server

python run.py --port 8765

Opens at http://localhost:8765. Paste your API key in Settings.

4

Start evaluating

Go to Evaluate or Matrix and start rating with J / K / L.
